#define INIT_GLOBAL_LOG_VERBOSITY ( verbosity)     honey::GLog::Get().SetGlobalVerbosity(honey::LogVerbosity::k##verbosity)

Initialize global log verbosity. If you set global log verbosity is Error, filtered the log in your code that log verbosity greater than global log verbosity. For example, LOG(Temp, Warn, "something warning"); will be not print.
